# Framer motion

a practice of [Shanu's Framer motion](https://github.com/iamshaunjp/framer-motion/tree/lesson-13) course

## Goals

1. Anmating elements
2. Initial Animation State
3. Transition Options
4. Hover Animations
5. Variants
6. Keyframes
7. Repeating Animations
8. Animate Presence
9. Animating Routes
10. Modal Animation
11. Animating SVG's
12. Creating a Loader
13. UseCycle Hook
14. Dragging Items
